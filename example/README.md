# UniBit API example

*A Javascript module example for UniBit API*

## Run Example
To run UniBit example, type:
```
cd your/path/to/unibit-js/example
```
then build example:
```
npm install
```
run example:
```
node index.js
```


## Develop On Local
### Pull 

```
git clone git@github.com:unibit-api/unibit-js.git
```

### Build

In root directory: 
```
npm install
npm link
```

### Install

```
npm install
npm link unibit-js
```
